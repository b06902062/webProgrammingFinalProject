# [109-1] Web Programming Final
## (Group 99) MuseOptimus

### Demo 影片: 附於貼文內
### 使用/操作方式: 點擊連結即可開始
### Project Goal:
1. 透過可調旋鈕讓使用者參與 AI 創作音樂的過程
2. 透過評分 & 推薦增加優秀創作的曝光度，
3. 讓使用者瞭解 AI 音樂的潛能，評分結果可作為模型訓練資料的 Crowdsourcing
### 使用工具:
#### Front end
  1. React
  2. HTML5 Canvas
  3. Soundfont Player
  4. Ant Design
  5. Material-UI
  6. Axios
#### Back end
  1. Python
  2. Flask
  3. PyTorch
  4. FluidSynth
#### Database
  1. MongoDB
  2. PyMongo
